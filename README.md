2019 华为软件精英挑战赛
=======================

![GitHub](https://img.shields.io/github/license/icgw/HUAWEI-CodeCraft-2019)

> 虽然结果不佳，但并不遗憾。

- 赛区：京津东北
- 团队：果树派
- 初赛：17名
- 复赛：20名（淘汰）

## 说明
1. 初版的判题器，对小规模数据误差较小，大规模数据效率低下。
   + 遗漏了某些规则，如：直行、左转、右转的优先次序。
   + 数据结构设计不全，且不合理。
   + 死锁的判定存在漏洞。
2. 判题器来不及完善，后期便不继续使用。
3. 求解器设计足够通用。
4. 初赛用了最短路径算法，效果拔群。然而，复赛的数据容易出现死锁。
